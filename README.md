# 📡 APRS-IS Passcode Generator

A simple, fast, and mobile-friendly **APRS-IS Passcode Generator** built with pure HTML, CSS, and JavaScript.

Generate your APRS-IS passcode instantly from your amateur radio callsign—no server, no tracking, and no data leaves your browser.

> **Live Demo:**  
> https://rn-ksheer.github.io/aprs-passcode-generator/

---

## ✨ Features

- ⚡ Instant APRS-IS passcode generation
- 📱 Responsive design for desktop and mobile
- 🌙 Dark & ☀️ Light theme
- 📋 One-click copy to clipboard
- 🔒 Runs completely in your browser
- 🚀 No installation required
- ❤️ Open Source

---

## 🚀 Usage

1. Open the web page.
2. Enter your amateur radio callsign.
3. Use only the **base callsign** (without SSID).

Example:

```
Correct : VU25RK
Incorrect: VU25RK-10
```

Your APRS-IS passcode is generated instantly.
